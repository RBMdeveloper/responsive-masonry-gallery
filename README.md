# Responsive Masonry Gallery

A fully responsive masonry-style image gallery with automatic layout handling for portrait and landscape images.

## Features

- Responsive masonry layout
- Automatically adjusts mixed image sizes
- Portrait + landscape support
- Lightbox popup preview
- Pure HTML, CSS & JavaScript
- No dependencies required
- Mobile friendly
- Custom scoped classes to avoid conflicts

## Installation

Copy the HTML, CSS, and JavaScript into your project.

```html
<section class="cg-gallery-section">
    <div class="cg-gallery-grid">

        <div class="cg-gallery-item">
            <img src="images/1.jpg" alt="">
        </div>

        <div class="cg-gallery-item">
            <img src="images/2.jpg" alt="">
        </div>

    </div>
</section>
```

---

## Responsive Breakpoints

| Screen Size | Columns |
|---|---|
| Desktop | 4 |
| Tablet | 3 |
| Mobile | 2 |
| Small Mobile | 1 |

---

## Lightbox Support

Clicking any image opens it in a fullscreen popup lightbox.
